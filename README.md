# Mob Grinding Utils

Mob Grinding Utils updated for **NeoForge 26.1.2**.

This repository is prepared for end users.

## Download

Get the compiled `.jar` from `dist/mob_grinding_utils-1.1.10+mc26.1.2.jar` in this repository.

## Requirements

- Minecraft `26.1.2`
- NeoForge `26.1.2.76`

## Installation

1. Install Minecraft and NeoForge for the versions above.
2. Download the mod `.jar` from Releases.
3. Place the `.jar` file in your `mods` folder.
4. Start the game.

## What is included

- Updated GUI and rendering for NeoForge 26.1.2
- Recipes and tags adjusted for better modpack compatibility
- English and Portuguese documentation

## Notes

- The repository is meant for players, pack makers, and servers.
- Development reference folders are not part of the final release.
