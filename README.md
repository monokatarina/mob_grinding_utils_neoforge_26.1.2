### O que e

Mob Grinding Utils adiciona blocos e itens para farms de mobs, processamento de XP, progressao com chicken feed, tanques, conveyors, fans e automacao relacionada.

Este fork atualiza o mod para NeoForge 26.1.2 e mantem as mudancas compativeis com modpacks modernos.

### Download

1. Clique aqui para baixar direto: <https://github.com/monokatarina/mob_grinding_utils_neoforge_26.1.2/releases/latest/download/mob_grinding_utils-1.1.10%2Bmc26.1.2.jar>

1. Instale o Minecraft e o NeoForge nas versoes acima.
2. Baixe o `.jar` do mod em Releases.
3. Coloque o `.jar` na pasta `mods`.
4. Abra o jogo.

### O que este fork inclui

- GUI e renderizacao atualizadas para NeoForge 26.1.2
- Recipes e tags ajustadas para melhor compatibilidade com modpacks
- Delightful Dirt ajustado para usar o mesmo ritmo de geracao do Dreadful Dirt, mantendo a diferenca: Dreadful gera monstros e Delightful gera criaturas passivas
- Ajustes anti-lag para Dreadful Dirt e Delightful Dirt: mobs gerados pela dirt nascem sem IA, somem mais rapido e respeitam limite de 50 mobs na area
- Slimes gerados pela Dreadful Dirt agora so podem nascer pequenos ou medios, nunca grandes
- Documentacao em portugues e ingles
- `.jar` pronto para download direto na Release

### Observacoes para jogadores

- Se voce ja usa um mundo antigo, faca backup antes de testar a nova versao.
- A pasta de codigo e para desenvolvimento; a release e o arquivo que a maioria dos usuarios deve baixar.

Mob Grinding Utils updated for **NeoForge 26.1.2**.

### What this is

Mob Grinding Utils adds utility blocks and items for mob farms, XP handling, chicken-feed progression, tanks, conveyors, fans, and related automation.

This fork updates the mod for NeoForge 26.1.2 and keeps the gameplay changes compatible with modern modpacks.

### Download

1. Click here to download directly: <https://github.com/monokatarina/mob_grinding_utils_neoforge_26.1.2/releases/latest/download/mob_grinding_utils-1.1.10%2Bmc26.1.2.jar>
2. Save the `.jar` file.
3. Put that `.jar` into your Minecraft `mods` folder.

If you are not used to GitHub, do not download the source code. Download only the `.jar` file using the direct link above.

### Requirements

- Minecraft `26.1.2`
- NeoForge `26.1.2.76`

### Installation

1. Install Minecraft and NeoForge for the versions above.
2. Download the mod `.jar` from Releases.
3. Move the `.jar` into your `mods` folder.
4. Start the game.

### Included in this fork

- Updated GUI and rendering for NeoForge 26.1.2
- Recipes and tags adjusted for better modpack compatibility
- Delightful Dirt now uses the same spawn timing as Dreadful Dirt, while keeping the intended difference: Dreadful spawns monsters and Delightful spawns passive creatures
- Anti-lag changes for Dreadful Dirt and Delightful Dirt: dirt-spawned mobs now spawn without AI, despawn faster, and respect a 50-mob area limit
- Slimes spawned by Dreadful Dirt can now only be small or medium, never large
- Portuguese and English documentation
- Release-ready `.jar` for direct download

### Notes for players

- If the mod does not show up, check that the `.jar` is inside the correct `mods` folder.
- If you are updating an older world, back it up before testing a new version.
- The source folder is for development; the release asset is the file most users should download.
